프로젝트에 docs 디렉토리를 만들고 docs/PRD.md 파일로 Notion CMS 활용 웹 프로젝트 PRD를 작성해줘

프로젝트 개요:
- 프로젝트명: 모바일 웨딩 청첩장
- 목적: 결혼식 초대장 및 정보를 다른 사람과 공유하기 위한 웹 페이지
- CMS 선택 이유: Notion에서 결혼식 정보를 작성 및 변경하면 자동으로 웹 페이지에 반영하기 위함

주요 기능:
1. Notion 데이터베이스에서 정보 가져오기
2. 결혼식장 뷔폐 메뉴를 볼 수 있는 기능
3. 반응형 디자인

기술 스택:
- Frontend: React.js 19, TypeScript
- CMS: Notion API (@notionhq/client)
- Styling: Tailwind CSS, Shadcn/ui
- Icons: lucide React
- Deployment: Vercel

화면 구성:
- 청접장 메인 세션
- 간단한 소개글 세션
- 신랑 신부 연락처 정보 세션
- 웨딩 사신 갤러리 세션
- 결혼식장 뷔폐 정보 세션
